SiteBuilder

  • Start

    • BEGINNER's GUIDE

  • ADD
    • Tag:

      • Select the element you want to add to the HTML body (Note: Select it before clicking the "Add" button).
    • Class:

      • Apply or change the class of an element.
    • Add:

      • Add an element to the HTML body.

      • Step 1:

        • Select the element name.

      • Step 2:

        • Click "ADD".

      • Step 3:

        • Next, click the parent element in the body where you want to insert the element.
    • Delete:

      • Delete an element from the HTML body.

      • Step 1:

        • Click "DELETE".

      • Step 2:

        • Next, click the element that you want to delete.
    • Exchange:

      • Exchange the positions of two elements or append an element from one parent node to another.

      • Step 1:

        • Click "EXCHANGE".

      • Step 2:

        • Now, click the element that you want to append.

      • Step 3:

        • Next, click the parent element to which you want to append the element.
    • Reset:

      • clear Inner Style of any Element.

      • Step 1:

        • Click "RESET".

      • Step 2:

        • Now, click the element that you want to clear.
  • SIZE
    • width:

      • set width of element.
    • height:

      • set height of element.
  • BORDER
    • Left:

      • Sets the left border of the element.
    • Right:

      • Sets the right border of the element.
    • Bottom:

      • Sets the bottom border of the element.
    • Top:

      • Sets the top border of the element.
    • Radius:

      • Sets the curvature of the element's border.
    • Style:

      • Sets the design (style) of the element's border.
    • Sizing:

      • Controls how the total width and height of an element are calculated. Values: content-box , border-box .
  • PADDDING
    • Left:

      • Sets the padding on the left side of the element.
    • Right:

      • Sets the padding on the right side of the element.
    • Bottom:

      • Sets the padding on the bottom side of the element.
    • Top:

      • Sets the padding on the top side of the element.
  • MARGIN
    • Left:

      • Sets the margin on the left side of the element.
    • Right:

      • Sets the margin on the right side of the element.
    • Bottom:

      • Sets the margin on the bottom side of the element.
    • Top:

      • Sets the margin on the top side of the element.
  • POSITION
    • Left:

      • Sets the left position of an element when it is positioned (using relative , absolute , fixed , or sticky ).
    • Right:

      • Sets the right position of an element when it is positioned (using relative , absolute , fixed , or sticky ).
    • Top:

      • Sets the top position of an element when it is positioned (using relative , absolute , fixed , or sticky ).
    • Bottom:

      • Sets the bottom position of an element when it is positioned (using relative , absolute , fixed , or sticky ).
    • Z-Index:

      • Controls the stacking order of positioned elements. Higher values are in front of lower values.
    • Float:

      • Positions an element to the left or right, allowing other content to wrap around it. Values: left , right , none .
    • Clear:

      • Controls the behavior of floated elements. Prevents elements from floating on the specified side. Values: left , right , both , none .
  • DISPLAY
    • Display:

      • Defines the display behavior of an element. Common values: block , inline , inline-block , none , flex , grid .
    • Grid-Gap:

      • Sets the space between rows and columns in a grid layout. Can be specified for both row and column gaps.
    • Grid-Template-Columns:

      • Defines the number and size of columns in a grid layout. Example: grid-template-columns: 100px auto 200px; .
    • Grid-Template-Rows:

      • Defines the number and size of rows in a grid layout. Example: grid-template-rows: 50px auto; .
    • Flex:

      • Specifies the flexibility of a flex item. A shorthand for flex-grow , flex-shrink , and flex-basis .
    • Flex-Direction:

      • Defines the direction of the flex items within a flex container. Values: row , row-reverse , column , column-reverse .
    • Flex-Wrap:

      • Specifies whether the flex items should wrap onto multiple lines. Values: nowrap , wrap , wrap-reverse .
    • Align-Content:

      • Aligns the flex container's lines within the container when there is extra space along the cross axis. Values: flex-start , flex-end , center , space-between , space-around , stretch .
    • Align-Items:

      • Aligns flex items along the cross axis (vertical if flex direction is row). Values: flex-start , flex-end , center , baseline , stretch .
    • Justify-Content:

      • Aligns flex items along the main axis (horizontal if flex direction is row). Values: flex-start , flex-end , center , space-between , space-around , space-evenly .
  • TEXT
    • Text:

      • Enter the text content in the textarea field.
    • Color:

      • Select a color for the text.
    • Opacity:

      • Adjust the opacity of the text (between 0 and 1).
    • Alignment:

      • Set the alignment of the text. Options are left , center , right , justify , start , or end .
    • Underline:

      • Select whether you want to apply an underline to the text. Options are none , overline , underline , or line-through .
    • Underline Color:

      • Choose the color for the underline.
    • Underline Opacity:

      • Set the opacity of the underline (between 0 and 1).
    • Underline Style:

      • Select the style for the underline. Options are solid , wavy , dashed , dotted , or double .
    • Underline Thickness:

      • Set the thickness of the underline. Enter a number value, such as 2px , 1em , etc.
    • Transform:

      • Transform the text. Options include none , uppercase , capitalize , and lowercase .
    • Wrap:

      • Set how text should wrap within the container. Options include stable , pretty , nowrap , wrap , and balance .
  • FONT
    • Size:

      • Set the font size using a number value. You can select a unit of measurement like px , em , rem , etc.
    • Family:

      • Set the font family. Enter a font name (e.g., "Arial", "Times New Roman") or a generic font family.
    • Style:

      • Choose the font style. Options include normal , italic , or oblique .
    • Weight:

      • Set the weight (boldness) of the font. Options include numeric values ( 100 , 200 , 300 , etc.) or keywords ( bold , bolder , lighter , or normal ).
    • Line Height:

      • Set the line height for the text. Enter a number value and select a unit like px , em , or rem .
  • TABLE
    • Add Rows:

      • Add new rows to the table by clicking the 'Add row' button.
    • Add Columns:

      • Add new columns to the table by clicking the 'Add column' button.
    • Remove Rows:

      • Remove rows from the table by clicking the 'Remove row' button.
    • Remove Columns:

      • Remove columns from the table by clicking the 'Remove column' button.
    • Border Collapse:

      • Set whether the table borders are collapsed or separated. Options: separate , collapse .

    • Table Padding:

      • Set the padding of the table cells. Enter a value and choose a unit such as px , em , etc.

    • Table Text Alignment:

      • Set the text alignment for the table cells. Options: left , center , right , justify , start , end .
    • Table Border:

      • Define the border of the table. Enter the size, style, and color (e.g., 2px solid black ).

    • Table Head:

      • Set the background color and text color for the table header.
    • Table Even Rows:

      • Set the background color and text color for even rows in the table.
    • Table Odd Rows:

      • Set the background color and text color for odd rows in the table.
  • BACKGROUND
    • Color:

      • Set the background color of an element. You can pick a color using a color picker, and adjust the opacity with a range slider.
    • Image:

      • Set the background image of an element by entering a color or a URL (link) for the image.
    • Size:

      • Define the background image's size by specifying the width and height with units (e.g., 50px 100px , 100% 100% ).
    • Position:

      • Set the position of the background image by specifying the X and Y coordinates with units (e.g., 50% 50% , left top ).
    • Repeat:

      • Set how the background image should repeat. Options include:

      • none:

        • No repetition.

      • repeat:

        • Repeats the image both horizontally and vertically.

      • space:

        • Repeats the image with space between them.

      • round:

        • Repeats the image, adjusting its size to fit the container.

      • repeat-x:

        • Repeats the image horizontally.

      • repeat-y:

        • Repeats the image vertically.

      • no-repeat:

        • Prevents the image from repeating.

    • BG Blendmode (optional):

      • Set the blending mode for the background layer. Options include:

      • none:

        • No blending.

      • color:

        • Blends the color values.

      • color-burn:

        • Darkens the background.

      • color-dodge:

        • Brightens the background.

      • darken:

        • Darkens the background.

      • difference:

        • Creates a difference effect.

      • exclusion:

        • Creates an exclusion effect.

      • hard-light:

        • Creates a contrast effect.

      • hue:

        • Applies the hue of the background.

      • lighten:

        • Lightens the background.

      • luminosity:

        • Adjusts the luminosity of the background.

      • multiply:

        • Multiplies the background color.

      • normal:

        • Standard blend mode (no blending).

      • overlay:

        • Combines both darken and lighten effects.

      • saturation:

        • Adjusts the saturation of the background.

      • screen:

        • Creates a lighten effect.

      • soft-light:

        • Creates a soft light effect.
    • Mix Blendmode (optional):

      • Set the mix blending mode for the element's content. Options are similar to the BG Blendmode options, such as: none , color , color-burn , multiply , normal , overlay , etc.

  • TRANSITION
    • Transition Duration:

      • Defines the duration of the transition. The value is typically given in seconds (s).

      • text:

        • A value representing the time duration (e.g., 1s , 0.5s ).
    • Transition Property:

      • Specifies which property should be transitioned. Common values include:

      • all:

        • Applies transition to all properties.

      • none:

        • No properties will be transitioned.
    • Transition Timing Function:

      • Defines the speed curve of the transition. The available options are:

      • linear:

        • The transition has a constant speed.

      • step-end:

        • The transition happens in steps, and it ends at the last step.

      • step-start:

        • The transition happens in steps, and it starts at the first step.

      • ease:

        • The transition starts slowly, accelerates, and then slows down again.

      • ease-in:

        • The transition starts slowly and accelerates.

      • ease-out:

        • The transition starts quickly and slows down.

      • ease-in-out:

        • The transition starts and ends slowly, with a faster middlesection.
    • Cursor:

      • Specifies the type of cursor to be displayed when hovering over the element. Available options include:

      • alias:

        • Represents an alias for a file or link.

      • none:

        • No cursor is shown.

      • crosshair:

        • A crosshair cursor.

      • default:

        • The default cursor.

      • grab:

        • Indicates the element is draggable.

      • grabbing:

        • Indicates the element is being dragged.

      • help:

        • A cursor indicating help is available.

      • move:

        • A cursor indicating the item can be moved.

      • pointer:

        • Indicates a link or clickable item.

      • progress:

        • Indicates that the application is busy, but the user can stillinteract.

      • text:

        • Indicates text can be selected.

      • wait:

        • Indicates that the application is busy and the user should wait.

      • col-resize:

        • Indicates the element can be resized horizontally.

      • row-resize:

        • Indicates the element can be resized vertically.

      • e-resize:

        • Indicates the element can be resized from the east side.

      • n-resize:

        • Indicates the element can be resized from the north side.

      • nw-resize:

        • Indicates the element can be resized from the northwest corner.

      • ns-resize:

        • Indicates the element can be resized from the north-south direction.

      • vertical-text:

        • Indicates vertical text input.

      • not-allowed:

        • Indicates an action is not allowed.

      • zoom-in:

        • Indicates the user can zoom in.

      • zoom-out:

        • Indicates the user can zoom out.
    • Visibility:

      • Defines the visibility of an element. The options include:

      • visible:

        • The element is visible.

      • collapse:

        • The element is collapsed, but it still takes up space.

      • hidden:

        • The element is not visible and does not take up any space.
    • Overflow X:

      • Defines the behavior of content overflow along the X-axis. The options include:

      • auto:

        • Displays scrollbars if the content overflows.

      • hidden:

        • Hides content that overflows the container.

      • scroll:

        • Always displays scrollbars, even if not needed.

      • visible:

        • The overflow content is visible beyond the container's bounds.
    • Overflow Y:

      • Defines the behavior of content overflow along the Y-axis. The options include:

      • auto:

        • Displays scrollbars if the content overflows.

      • hidden:

        • Hides content that overflows the container.

      • scroll:

        • Always displays scrollbars, even if not needed.

      • visible:

        • The overflow content is visible beyond the container's bounds.
    • clip-path

      • Defines the behavior of content overflow along the Y-axis. The options include:

      • circle():

        • Creates a circular clipping region. The circle is defined by a radius and an optional center point.

      • ellipse():

        • Creates an elliptical clipping region, defined by horizontal and vertical radii and an optional center point.

      • polygon():

        • Creates a polygonal clipping region defined by a set of coordinates.

      • path():

        • Allows for a custom clipping path using SVG path syntax to define complex shapes.

      • inset():

        • Creates a rectangular clipping region with optional padding around the content, effectively "cutting" into the element.
